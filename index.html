프로젝트 개요 개발내용
기간 : 2023.04.17 ~ 2023.05.12 (4주)
인원 : 2명
역할 : 팀장


첫 팀 프로젝트는 커뮤니티 사이트를 구현해보고 싶었습니다.
팀원과의 협의 후 역할 분담을 정확하게 나누고 진행하였습니다. 		- 개인 이메일 인증과 기타 유효성 검사를 통한 회원가입
- 여러 주제의 자유게시판, 사진 위주의 상품 후기게시판
- 회원제 신발 응모 이벤트
- 크롤링을 사용해 발매 신발 정보와 여러 소식을 접함
- 회원간 소통이 가능한 쪽지 기능
담당 업무
담당 업무 : 형상관리, db설계, 전체 페이지 디자인, 게시판, 댓글, 발매정보/발매 소식

  1. 디자인
    • BootStrap5
      - 부트스트랩5 문서를 참고하여 시맨틱웹을 지향해 디자인 하였습니다.
    • Jquery
      - jquery를 이용해 여러 요소를 동적으로 구현하였습니다. (이미지 슬라이드, 페이지 맨 위로, 이미지 다음/이전 버튼, 대댓글폼 on/off, 쪽지함 이동 버튼 등)
  2. DB설계
    • Maria DB
      - 각 테이블간의 연관성을 고려하여 외래키관계와 제약조건(연쇄삭제, 변경)을 설정하였습니다.
  3. 게시판
    • 게시글 리스트
      - 회원 테이블과 댓글 테이블을 조인하여 작성자에 프로필사진과 게시글 제목에 댓글 수를 표시하였습니다.
      - jstl 조건문을 사용하여 제목이 길었을 때, 사진 첨부 여부, 인기 게시물 여부, 오늘 작성한 글인지 체크하여 그에 맞는 UI를 구현하였습니다.
      - 후기게시판의 경우 컨트롤러에서 img태그의 src값을 정규식으로 추출하여 썸네일로 이용하였습니다. 사진 개수도 체크하여 게시글의 몇 장의 사진이 있는지도 표기하였습니다.
    • MyBatis 동적쿼리를 이용해 게시글 필터링
      - 인기 글과 전체 글을 구분하여 볼 수 있게 구현하였습니다.
      - 제목+내용 또는 작성자로 검색할 수 있게 구현하였습니다.
      - 페이지 번호를 파라미터로 넘겨서 페이지에 맞는 게시글들이 나오도록 구현하였습니다.
    • 게시글 공개/비공개
      - 게시글 각각의 체크박스를 만들고, jstl c:set 변수에 게시글 번호를 반복문을 돌면서 추가하여 체크된 게시물 번호만 컨트롤러에서 넘겨 db에서 공개 여부 값을 참으로 만들어 처리하였습니다.
    • 이전글/다음글
      - 현재 게시물 번호를 파라미터로 받아 연관서브쿼리를 사용하여 이전 게시물의 번호와 다음 게시물의 번호를 리턴하여 그 게시물로 이동할 수 있게 구현하였습니다.
    • 게시글 추천
      - 회원만 추천 가능하게 조건설정을 하였고, 추천 버튼을 누르면 ajax를 이용하여 게시물 번호와 회원의 닉네임을 파라미터로 받아 추천 테이블로 insert 하였고, board 테이블의 추천 수는 1이 오르도록 update 하였습니다.
  4. 댓글
    • 댓글 작성폼/대댓글 작성폼
      - 회원만 작성 가능하며, 현재 로그인한 회원의 닉네임과 프로필 사진이 댓글 작성폼에 보이게 구현하였습니다.
      - 댓글마다 댓글 작성 폼을 만들어 숨겨놓고 대 댓글 작성 버튼을 누르면 해당 폼이 밑으로 보이게 구현하였습니다.
    • 댓글작성/댓글추천
      - 댓글작성과 추천시 ajax를 페이지번호와 기타 파라미터값을 받아서 db에 등록 후 페이지 이동없이 본인이 작성한 위치에서 바로 보이게 구현하였습니다.
  5. 발매정보/발매소식
    • jsoup 크롤링
      - '럭키드로우'라는 실시간 발매정보 홈페이지를 크롤링하여 이용자에게 필요한 정보의 태그와 클래스 명을 크롤링하여 객체에 저장하고 모델을 만들어 view쪽으로 넘겨서 구현하였습니다.
    • 검색
      - 크롤링하는 과정에서 String 참조변수의 contains 메서드를 이용하여 제목과 발매처를 파라미터로 받아 필터링하였습니다.

숫자야구게임

프로젝트 개요
기간 : 2023.03.27 ~ 2023.03.30 (4일)
인원 : 1명


백준알고리즘에서 풀었던 숫자야구문제를 참고하여 게임을 제작하였습니다.
개발내용
  1. 기능
    • 사용자 입력 중복숫자 체크 / 0 체크 함수
      - 사용자 입력값을 split함수를 사용하여 배열로 만든 후 반복문을 통해 0이 있거나 또는 같은 숫자가 있는지 중복체크를 하였습니다.
    • 채팅창 li태그 생성 함수
      - 사용자 입력값과 컬러문자열을 매개변수로 받아 li태그를 생성해서 채팅 ul태그에에 append하였습니다.
    • 생성된 li태그 자동 포커싱 함수
      - 채팅 ul태그에 마지막 li를 가져와서 포커싱하고 ul.scrollTo 함수를 사용해 스크롤을 내리는 함수입니다.
    • 숫자게임의 타이머 함수
      - 밀리초 변수에 100을 넣어서 인터벌함수를 사용해 1씩 내려서 0이되면 1초가 카운트 되도록 이중으로 타이머를 설정하였습니다.
      - 남은 시간초에 따라 조건을 부여하여 메세지가 출력되도록, 중간에 답을 맞추게 된다면 타이머클리어 함수를 사용해 종료되도록 구현하였습니다.
    • 사용자가 입력한 숫자를 분석하는 함수
      - 시스템값과 사용자입력값을 각각 split하여서 배열에 담고 이중 for문을 사용하여 인덱스를 활용해 자릿수와 숫자가 같으면 strike 카운트를 올리고 숫자만 같으면 ball 카운트를 올립니다.
